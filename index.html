Brocli

This server is running Brocli, a simple Javascript application that reads data stored in the consensus memory (RAM) of an Antelope blockchain, following the GPPS standard.

This installation is connected to the UX Network blockchain, and is using the GPPS contract deployed to the datastoreutx smart contract account.

Try accessing the demo page, which is stored on and retrieved from UX Network RAM, and is decompressed and cached by this website.
